## Inter-VLAN Routing (Router-on-a-Stick)

Configured inter-VLAN routing using a router-on-a-stick setup. 
Created VLANs, configured trunking between the switch and router, and used subinterfaces to enable communication between networks.

### Key Skills:
- Router subinterfaces and 802.1Q encapsulation
- Trunk port configuration
- IP addressing across multiple VLANs
- Default gateway configuration

### What I Learned:
This lab provided hands-on experience enabling communication between segmented networks, reinforcing routing concepts and VLAN architecture used in real-world environments.
