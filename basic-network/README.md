## Basic Network Configuration (LAN Setup)

Built a simple LAN in Cisco Packet Tracer using two end devices and a switch. 
Configured manual IP addressing and verified connectivity using ICMP (ping).

### Key Skills:
- IP addressing and subnetting fundamentals
- End-to-end connectivity testing
- Understanding of Layer 2 switching

### What I Learned:
This lab reinforced how devices communicate within the same subnet and how to validate connectivity using basic network troubleshooting commands.
