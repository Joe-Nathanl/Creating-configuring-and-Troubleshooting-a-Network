## VLAN Configuration and Network Segmentation

Implemented VLANs on a switch to segment network traffic between multiple departments. 
Assigned switch ports to specific VLANs and verified isolation between networks.

### Key Skills:
- VLAN creation and management
- Switch port configuration
- Network segmentation

### What I Learned:
This lab showed how VLANs improve security and organization by isolating broadcast domains and controlling traffic between groups.
